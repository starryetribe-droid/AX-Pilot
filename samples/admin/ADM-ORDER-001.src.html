UI 요소

  • 1. 상단 유틸리티 바
       • 좌: DESIGN MEAL 로고 + Admin Mode
       • 우: 관리자(admin) · Dark Mode · View Site · Log Out

  • 2. 좌측 GNB
       • 대분류 14개(CRM~이력), 접이식
       • 활성: 주문관리 → 하위 7개(주문관리 선택)

  • 3. 페이지 헤더
       • 타이틀 "주문상세" + 주문번호
       • 우측 액션: 목록 이동, 저장

  • 4. 주문 정보
       • 주문자/주문상태/주문일시
       • 결제수단/결제일시/외부몰 주문번호
       • 판매채널/장바구니쿠폰/맞춤여부
       • 전화주문여부/정기결제번호

  • 5. 배송지
       • 수령자/연락처/배송지 주소/요청사항/출입비밀번호

  • 6. 메모
       • 관리자용 메모 입력(textarea)

  • 7. 결제 요약
       • 상품금액/주문합계/배송비
       • 상품별·결합·장바구니·포인트 할인(할인=빨강)
       • 결제금액(볼드) + 하단 강조 박스

  • 8. 액션 버튼
       • 전체취소 / 변경로그

  • 9. 안내문
       • 녹즙 상품 주문 변경·부분취소 정책 안내(빨강)

  • 10. 주문상품 테이블
       • 배송구분/배송장소/송장번호/sub주문번호/상품코드
       • 주문상품/기간/패턴/1회수량/정상가/쿠폰비/실결제금액
       • 행 체크박스(부분취소 선택) + 주문상태 pill

동작 / 예외

  • 전체취소
       • 결제완료 등 취소 가능 상태에서만 활성

  • 변경로그
       • 주문 변경 이력 모달/팝업 노출

  • 부분취소
       • 테이블 행 체크 후 부분취소 (녹즙=하이톡 주문 확정 이후 가능)
