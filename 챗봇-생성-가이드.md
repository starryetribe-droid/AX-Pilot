# 디자인밀 AI 챗봇 화면, 이렇게 만들어요 🙂

> **처음 써보셔도 괜찮아요.** 코딩 지식 없이, Claude에게 한국어로 부탁하면 됩니다.  
> 엑셀(인텐트 정의서)에 적어 둔 시나리오를 → 실제 챗봇처럼 보이는 **화면 그림**으로 자동으로 만들어 줍니다.

이 문서 한 장이면 **준비 → 만들기 → 확인**까지 끝낼 수 있어요.

## 1. 시작 전에 (딱 두 가지만) ✅

1. **엑셀 파일이 어디 있는지** 알기 — "AI챗봇 인텐트정의" 엑셀이 컴퓨터 어딘가에 있어야 해요. 위치만 알면 됩니다.
2. **도구 설치** — 이미 돼 있으면 그냥 쓰면 돼요. 안 돼 있으면 Claude 채팅창에 아래 문장을 붙여넣으세요. (한 번만)

   ```
   챗봇 화면 도구 설치 도와줘
   ```

   설치가 끝나면 **새 채팅을 한 번 시작**하세요. 그래야 새 기능이 인식됩니다.

## 2. 화면 만들기 (이게 전부예요) ✏️

Claude 채팅창에 이렇게 적으세요. **숫자만 바꾸면** 다른 시나리오도 만들어집니다.

> **인텐트 9번 챗봇 화면 만들어줘**

처음엔 Claude가 **작성자 이름**이나 **엑셀 위치**를 물어볼 수 있어요. 알려주기만 하면 나머지는 알아서 합니다.

잠깐 기다리면 "다 만들었어요"라고 알려줘요. 끝! 🎉

## 3. 결과 확인하기 👀

만들어지면 컴퓨터에 폴더가 하나 생겨요. 보통 이 위치예요:

```
다운로드 → SB_챗봇 → IT009      (IT 뒤 숫자가 인텐트 번호: 9번이면 IT009)
```

그 폴더 안의 **`index.html` 더블클릭** → 챗봇의 단계별 화면이 한 페이지에 쭉 보여요(크롬 등에서 열림).  
"방금 만든 화면 폴더 열어줘"라고 Claude에게 부탁해도 됩니다.

## 5. 화면은 이런 조각들로 그려져요 🧩

챗봇 답변은 아래 조각(컴포넌트)들을 조합해 그려집니다. **무엇을 쓸지는 엑셀에 이미 정해져 있어서**, 여러분이 고를 필요는 없어요. 자동으로 알맞게 그려집니다.

**텍스트** — 챗봇이 말로 안내하는 부분
- **텍스트말풍선** — 챗봇 발화 텍스트를 표시하는 기본 말풍선

**선택** — 사용자가 버튼으로 고르는 부분
- **선택버튼** — 2~4개 선택지를 제시하는 버튼 그룹

**카드** — 정보를 카드 모양으로 보여주는 부분
- **단건카드** — API 조회 결과 단건 데이터를 카드로 표시
- **카드리스트** — API 조회 결과 다건을 카드 형식으로 나열, 선택 가능
- **요약카드** — 실행 직전 정보 요약 표시
- **추천카드** — 상품·콘텐츠 추천 가로 스크롤 카드
- **이관카드** — 상담사 연결 안내 카드

**입력** — 사용자가 날짜·수량·글자를 직접 넣는 부분
- **입력위젯-날짜** — 날짜 선택 캘린더 또는 빠른 날짜 버튼
- **입력위젯-수량** — 수량 조정 위젯
- **입력위젯-텍스트** — 자유 텍스트 입력 필드
- **결제위젯** — PG 연동·인증 등 결제 보안 영역

**배너** — 처리 중·완료·오류 상태를 알려주는 부분
- **로딩인디케이터** — 처리 대기 상태 표시 + 안내 메시지
- **결과배너** — 처리 완료·실패 결과 표시
- **에러배너** — 오류·예외 상황 안내

**액션** — 다음 행동(버튼·이동)을 안내하는 부분
- **액션버튼** — 결과 화면의 다음 행동 안내 버튼
- **이동링크** — 앱 내 다른 페이지·딥링크 이동 안내 버튼

## 6. 답이 갈릴 때는? (분기) 🔀

"성공/실패"처럼 결과가 갈리는 경우엔, **화면을 따로 나누지 않고** 해당 조각 옆에 두 경우를 **나란히** 보여줘요.

- **결과배너**: 성공 / 실패 를 나란히
- **로딩인디케이터**: 기본 / 취소 가능 를 나란히
- **선택 버튼·입력칸**은 항상 **안내 말풍선과 함께** 보여줘요(버튼만 덩그러니 두지 않아요).

## 7. 잘 안 될 때 🛟

| 이런 경우 | 이렇게 하세요 |
|------------|---------------|
| 화면이 비어 있어요 | 그 번호는 아직 준비 안 된 시나리오예요. **4번 표**에서 골라 보세요. |
| 엑셀을 못 찾는대요 | 엑셀이 저장된 **정확한 위치**를 다시 알려 주세요. |
| 노란색 "로고 경고"가 떠요 | **무시해도 됩니다.** 화면 그림엔 영향 없어요. |
| 설치가 안 돼 있대요 | 채팅창에 `챗봇 화면 도구 설치 도와줘` 를 붙여넣으세요. |
| 그래도 막혀요 | Claude에게 평소처럼 물어보세요: "인텐트 16번 만드는 법 알려줘" |

## 8. 컴포넌트 가이드도 만들 수 있어요 📚

화면 말고, **어떤 조각들이 있는지 한눈에 보는 가이드**도 만들 수 있어요. Claude에게:

> **챗봇 컴포넌트 가이드 만들어줘**

그러면 두 가지 형태가 `다운로드 → SB_챗봇 → _GUIDE` 에 생겨요:

- **웹 카탈로그**(`component-guide.html`) — 조각마다 미리보기 + 상세 표
- **SB 양식**(`component-guide-sb.html`) — 한 화면에 모든 조각을 모아 둔 ETRIBE 와이어프레임
